# DateTime and Random Practice Tasks

This folder contains 13 practice tasks covering Python's `datetime` and `random` modules.

## 📅 DateTime Tasks (1-5)

### Task 1: Today's Info Program
**File:** `Task_1_todays_info.py`
- Print today's date
- Print current time
- Print day name (Monday, Tuesday, etc.)

**Key Concepts:**
- `datetime.now()`
- `strftime()` formatting
- Date and time formatting

---

### Task 2: Age Calculator
**File:** `Task_2_age_calculator.py`
- Calculate age in years from date of birth
- Calculate total days lived

**Key Concepts:**
- `strptime()` for parsing dates
- Date subtraction
- Age calculation logic

---

### Task 3: Days Left for Exam
**File:** `Task_3_days_left_for_exam.py`
- Calculate days remaining until exam
- Check if exam date is past or future

**Key Concepts:**
- Date comparison
- Date arithmetic
- Conditional logic with dates

---

### Task 4: Digital Clock
**File:** `Task_4_digital_clock.py`
- Create a live updating digital clock
- Updates every second

**Key Concepts:**
- `while True` loop
- `time.sleep()`
- Real-time updates
- Keyboard interrupt handling

---

### Task 5: Greeting System
**File:** `Task_5_greeting_system.py`
- Time-based greetings:
  - Morning (5-12)
  - Afternoon (12-17)
  - Evening (17-21)
  - Night (21-5)

**Key Concepts:**
