## 🚀 Getting Started

### Prerequisites
- Python 3.6 or higher
- Basic understanding of Python syntax

### How to Use
1. Navigate to the desired topic folder
2. Open any `.py` file
3. Run the file: `python filename.py`
4. Follow the prompts and input requirements

### Example
```bash
# Run a basic if-else question
cd "If-Else/Basic Checks"
python 1_positive_or_negative.py

# Run a loop question
cd "For Loop/For starting 20"
python 1_first_n_natural_number.py
```
